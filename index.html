Hydrix-C

Simplifying C++
Development

A C++ Operating system built off the Limine protocol made to demonstrate the
capabilities of LIBHYDRIX. It also can serve as a great tutorial for beginners

LIBHYDRIX is a library made for C++ using the Limine protocol. It is a library that aims to
provide a simplistic operating system development experience. It's loaded with all tools
neccessary for building an operating system. LIBHYDRIX is completely open-source and x64
